### 1.5 BMP文件存储格式

- BMP文件组成
  
  - 位图文件头结构(BITMAP**FILE**HEADER)
  
  - 位图信息头结构(BITMAP**INFO**HEADER)
  
  - 位图颜色表(RGBQUAD)
  
  - 位图像素数据

- BMP文件头
  
  ```c
  typedef struct
  {
      WORD    bfType;    //字节"BM"
      DWORD    bfSize;    //文件总字节
      WORD    bfReservrd1;//0
      WORD    bfReservrd1;//0
      DWORD    bfOffBits;// 位图数据距文件头的偏移量
  }BITMAPFILEHEADER
  ```

- 位图信息头
  
  ```c
  typedef struct
  {
       DWORD biSize; //本结构大小的字节数
       LONG biWidth; //单位像素点
       LONG biHeight; //单位像素点
       WORD biPlanes; //1
       WORD biBitCount； // 像素的位数(1/4/8/24)
       DWORD biCompression; // 压缩方式(0表示不压缩)
       DWORD biSizeImage; // 位图数据的字节数
       LONG biXPelsPerMeter; // 水平分辨率(像素点/米)
       LONG biYPelsPerMeter; // 垂直分辨率(像素点/米)
       DWORD biClrUsed; // 图像中使用的颜色数
       DWORD biClrImportant; // 图像中重要的颜色数
  }BITMAPINFOHEADER;
  ```

- 颜色表
  
  ```c
  typedef struct
  {
      BYTE rgbBlue;// 蓝色
      BYTE rgbGreen; // 绿色
      BYTE rgbRed; // 红色
      BYTE rgbReserved;// 0
  } RGBQUAD;
  ```

- 位图信息
  
  ```
  struct 
  {
       BITMAPINFOHEADER bmiHeader;
       RGBQUAD bmiColors[1]; // 颜色表
  }BITMAPINFO;
  ```

- 256位图文件调色板
  
  - 调色板 256项
  
  - 所占字节数:256*4=1024
  
  - 文件头占3行，每行16字节，可以存储4个表项，64*4=256个表项
  
  - 1024/16=64行
  
  - 总行数64+3=67
