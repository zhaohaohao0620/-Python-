## make

### 预定义变量
| 变量 | 含义 |
| ----- |:------:|
| AR | 库文件维护程序的名称,ar |
| ARFLAGS | 库文件维护程序的选项 |
| AS | 汇编程序的名称,as |
| ASFLAGS | 汇编程序的选项 |
| CC | C编译器名称,cc |
| CFLAGS | C编译器选项 |
| CPP | C预编译器, $(CC)-E |
| CPPFLAGS | C预编译器选项 |
| CXX | C++预编译器,g++ |
| CXXFLAGS | C++预编译器选项 |
| FC | FORTRAN编译器,f77 |
| FFLAGS | FORTRAN编译器选项 |
| RM | 删除文件,rm -f |

### 常见自动变量
| 变量 | 含义 |
| ----- |:------:|
| $* | 不包含扩展名的目标文件名称 |
| $+ | 所有的依赖文件 |
| $< | 第一个依赖文件的名称 |
| $? | 所有时间戳比目标文件晚的依赖文件 |
| $@ | 目标文件的完整名称 |
| $^ | 所有不重复的依赖文件 |
| $% | 如果目标是归档文件,则该变量表示目标的归档成员名称 |
